This project is a backend microservices demo that simulates a simple video streaming platform (e.g., Netflix or TV360) at a conceptual level. The system focuses on managing video metadata and exposing RESTful APIs to retrieve and manipulate that data.

The purpose of this project was educational: folder Java help me learn Java, while folder SpringBoot help me learn Spring Boot, and basic backend architecture through hands-on implementation during my internship at Viettel. The project simulates a video streaming backend using REST APIs consisting of services that perform functions such as login, registration, and get metadata data of video from MySQL database. In particular:
- Manage user login and registration with JWT authentication.
- Manages video metadata (e.g., title, description, duration, category).
- Supports basic CRUD operations on video data.
- Demonstrates microservice-style backend design. Allows API testing via Postman.

Technologies Used:
Java, Spring Boot (REST API development), MySQL (relational database for video metadata), Postman (API testing), Maven (dependency management)
